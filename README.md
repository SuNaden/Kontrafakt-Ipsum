# Kontrafakt Ipsum

![Ide Paťo](./icon.png)

Alfred workflow to generate lorem ipsum based on Kontrafakt lyrics. Recommended by 5 out of 5 Paťos.

### Usage
Usage in Alfred that generates the first 40 words of a random Kontrafakt song.
```
kntrfkt 40
```
